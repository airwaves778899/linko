# Code of Conduct

本專案採用 [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) 作為社群行為準則。

## 簡要原則

- **友善**：用尊重、有耐心的態度對待每一位參與者
- **包容**：歡迎不同背景、經驗、語言、技術程度的貢獻
- **建設性**：批評對事不對人；focus on the work, not the person
- **誠實**：開誠布公討論技術選擇與權衡；不要 passive-aggressive

## 不被接受的行為

- 任何形式的騷擾、歧視、人身攻擊
- 揭露他人個資（doxxing）
- 政治 / 宗教 / 種族議題的挑釁
- 持續打斷、刷屏、灌水

## 回報

如果你目擊或經歷不當行為，請聯絡 maintainer：
